# PRAM მოდელი
# 4.2 PRAM მოდელის ტიპები
- 4.2.1 EREW (Exclusive Read Exclusive Write)
```ერთდროულად მხოლოდ ერთ პროცესორს შეუძლია წაკითხვა ერთდროულად მხოლოდ ერთ პროცესორს შეუძლია ჩაწერა - ყველაზე უსაფრთხო```
- 4.2.2 CREW (Concurrent Read Exclusive Write)
``` რამდენიმე პროცესორს შეუძლია ერთდროულად წაკითხვა ჩაწერა მხოლოდ ერთს შეუძლია - უფრო სწრაფი, პრაქტიკასთან ახლოს```
- 4.2.3 CRCW (Concurrent Read Concurrent Write)
```ერთდროულად შეიძლება წაკითხვაც და ჩაწერაც აქ ჩნდება კონფლიქტის გადაწყვეტის წესები: Common — ყველა წერს ერთსა და იმავე მნიშვნელობას Arbitrary — ერთ-ერთი პროცესორი ირჩევა შემთხვევით Priority — პრიორიტეტიანი პროცესორი წერს - ყველაზე ძლიერი მოდელი ```
